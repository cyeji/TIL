## 웹 어플리케이션 이해

## 웹 서버 (Web Server)
- HTTP 기반으로 동작
- 정적 리소스 제공, 기타 부가기능
- 정적(파일) HTML,CSS, JS, 이미지, 영상
- Niginx, APACHE

## 웹 어플리케이션 서버(WAS - WebApplication Server)
- HTTP 기반으로 동작
- 웹 서버기능 + (정적 리소스 제공 기능)
- 프로그램 코드를 실행해서 애플리케이션 로직 수행
  - 동적 HTML, HTTP API(JSON)
  - 서블릿 , JSP, 스프링 MVC
  - 톰캣, Jetty, Undertow
